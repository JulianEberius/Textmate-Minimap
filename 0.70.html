TextMate-Minimap version 0.7 release notes

  • Improved minimap scrolling performance
  • Minimap can now be scrolled independently of the textview. Use the mouse wheel when hovering over the minimap.
  • The minimap can now be shown in a sidebar instead of in a drawer. This makes it fit in for people using ProjectPlus or MissingDrawer to turn their project drawer into a sidebar.
  • When opening the minimap, the window size will now be reduced to fit the minimap on the screen, if necessary (just like when opening the project drawer).
